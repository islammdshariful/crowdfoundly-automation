Prerequisite:
 
1. Java 
2. IntelliJ
3. Maven
4. TestNG 
5. AutoIT

Installation: 

1. Add TestNG jar in project
2. Check your chrome browser version and add the respective driver to the ./driver/ folder. Rename it "chromedriver.exe".
3. Build the project
4. Now Run Test Cases from ./src/main/java/testcases
